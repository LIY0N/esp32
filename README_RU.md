# ESP32 RPM to CAN Bus Converter

Этот скетч считывает сигнал с датчика оборотов (RPM) через прерывания, обрабатывает его и отправляет данные по шине CAN.

## Аппаратные требования

- ESP32 (или совместимая плата)
- Датчик оборотов (энкодер или импульсный датчик)
- CAN трансивер (например, MCP2515 или TJA1050)
- Питание 3.3В

## Подключение

### Датчик RPM
- Подключите датчик оборотов к пину 10 (GPIO10) на ESP32

### CAN шины
- CAN_TX (передача) - GPIO3
- CAN_RX (прием) - GPIO2

Подключите эти пины к соответствующим контактам CAN трансивера.
